Endpoints Disponibles

GET /webhook

Petición de tipo GET, enviar en URL: mode, token y challenge

Ejemplo de Request (Reemplazar VERIFY_TOKEN por el Token a enviar)

  • curl -X GET "https://meta-apis-consumer.herokuapp.com/webhook?hub.verify_token=VERIFY_TOKEN&hub.challenge=CHALLENGE_ACCEPTED&hub.mode=subscribe"

    • POST /webhook

    Petición de tipo POST, recibe un mensaje

    Ejemplo de Request

  • curl -H "Content-Type: application/json" -X POST "https://meta-apis-consumer.herokuapp.com/webhook" -d '{"object": "page", "entry": [{"messaging": [{"message": "TEST_MESSAGE"}]}]}'
